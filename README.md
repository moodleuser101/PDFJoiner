# PDFJoiner
Simple utility application for joining two or more source PDF files into a single PDF.

## Installtion
Download the latest release package (right-hand side of GitHub repository interface) and find the `PDFJoiner.jar` file. This can be deployed directly as an executable JAR file by any computer with a functioning Java Runtime Environment (JRE) installed that supports version 8 or greater of Java (i.e. JRE 1.8+). An example command line launch of the program would be:

```
java -jar PDFJoiner.jar
```
